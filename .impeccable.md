# Impeccable Design System

## Design Context

### 用户
- **身份**：开发者、像素艺术爱好者
- **使用场景**：快速创建图标、像素艺术创作
- **核心诉求**：简洁高效的工具，不分散注意力

### 品牌个性
- **关键词**：极简、精准
- **情感目标**：专业感、高效率、清晰直观

### 美学方向
- **视觉风格**：极简干净，受 Figma 影响
- **主色调**：亮色模式为主
- **色彩体系**：
  - 背景色：浅灰 (#fafafa, #f4f4f5)
  - 强调色：橙色点缀（替代当前的 indigo #6366f1）
  - 边框/分割：细线、中性灰
- **字体**：系统字体栈，清晰易读

### 设计原则
1. **功能优先** — 每个元素必须有明确用途，无装饰性元素
2. **留白有度** — 使用充足间距创造呼吸感，避免拥挤
3. **一致性** — 统一的圆角、间距、阴影规范
4. **清晰反馈** — 交互状态（hover/active/disabled）有明显区分
5. **克制用色** — 限制主色调数量，用橙色点缀强调关键操作

### 视觉规范（待实施）
- 圆角：4px（小元素）、8px（卡片/面板）、12px（模态框）
- 间距基准：4px 倍数（4, 8, 12, 16, 24, 32）
- 阴影：极轻阴影或无阴影，保持极简感
- 图标：使用 lucide-react 图标库，替代当前 emoji
