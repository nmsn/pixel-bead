# Pixel Bead 4 步流程重构设计方案

## 概述

将现有功能重组为 4 步流程：上传转换 → 编辑画布 → 背景效果 → 导出。步骤可通过顶部导航栏点击跳转。

## 整体布局

```
┌─────────────────────────────────────────────────┐
│  [1] ──── [2] ──── [3] ──── [4]    (居中，短)   │
│  上传   编辑   背景   导出                      │
├─────────────────────────────────────────────────┤
│  工具栏: [上传图片] [撤销] [重做]                │
├─────────┬───────────────────────────────────────┤
│         │                                       │
│  侧边栏  │          主内容区                    │
│  (工具)  │     (当前步骤内容)                   │
│         │                                       │
└─────────┴───────────────────────────────────────┘
```

- 顶部步骤条：居中显示，步骤数字可点击跳转
- 主界面默认进入步骤 2（编辑画布）
- 工具栏放置「上传图片」按钮，点击弹出步骤 1 弹窗

---

## 步骤 1：上传弹窗

**触发方式**：点击工具栏「上传图片」按钮，弹出模态框。

### 弹窗内容

| 区域 | 说明 |
|------|------|
| 上传区 | 拖拽/点击上传图片，显示图片预览和尺寸 |
| 网格选择 | 16×16 / 32×32 / 64×64 / 128×128 / 256×256 / 自定义 |
| 转换按钮 | 上传图片后可点击，转换为像素画布并关闭弹窗 |

---

## 步骤 2：编辑画布

现有主要功能区域，包含画笔、橡皮擦、填充、取色等工具，以及调色板。

---

## 步骤 3：背景效果

### 左侧设置面板

#### 背景类型

- **纯色**：单色选择器
- **渐变**：多颜色节点 + 渐变预览条 + 圆形角度调节器

#### 渐变配置

- 色标节点：可添加多个颜色（点击 `+` 增加节点）
- 渐变预览条：实时显示当前渐变效果
- 角度调节器：圆形表盘，指针从圆心出发指向当前角度；角度值「135°」显示在右下角外侧

#### 其他设置

- **圆角**：滑块控制，0-64px
- **顶部高光**：开关 + 强度滑块（0-100%）

### 右侧预览

实时显示背景效果 + 像素画布的合成预览。

---

## 步骤 4：导出

### 布局比例

左侧占 1 份，右侧占 2 份。

### 左侧：格式选择

| 格式 | 说明 |
|------|------|
| ICO | Windows 图标，包含 16/32/48/256 px |
| ICNS | macOS 图标，包含 16-1024 px |
| PNG | 多尺寸可选（8/16/32/48/64/128/256/512） |

### 右侧：效果预览

- 大尺寸预览图（180×180）
- 配置标签显示当前设置：渐变角度、高光强度、圆角值

---

## 技术实现要点

1. **状态管理**：统一的状态管理多步骤流程，步骤之间可自由跳转
2. **背景渲染**：支持纯色、渐变色（多节点可扩充）、光泽效果
3. **导出合成**：背景层在下，像素画布层在上，合并后导出
4. **圆形角度调节器**：拖拽旋转交互，指针从圆心出发
